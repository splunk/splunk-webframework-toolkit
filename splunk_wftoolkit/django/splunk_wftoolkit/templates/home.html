Splunk Web Framework Toolkit

Welcome to the Splunk Web Framework Toolkit! This app contains a collection of examples that show you how to use the components of the Web Framework. You'll find documentation and reference information, along with code templates to help you get started. You can view all of the examples in action in this app, but then be sure to look at the source code in /splunk_wftoolkit/django/splunk_wftoolkit/ to see how it's done.

Here's what's included:

Getting Started

Provides an overview of the basic concepts of the Web Framework.

Web Framework Components

Demonstrates all of the search managers and Splunk views that are included in the Web Framework, and includes reference information and code examples in JavaScript and Django.

Toolkit Components

Contains a collection of custom visualizations based on open-source libraries. These views extend SimpleSplunk view to connect to your Splunk server and work with your data.

Examples

Contains example dashboards that demonstrate how to work with different features of the Web Framework, including how to use tokens for data binding, add interactivity, work with events, and more.

Layout Templates

Provides page templates that show you how to lay out the content of your pages using the Splunk dashboard styles. You can use these layout templates as a starting point for your apps.

More Info

Contains useful links to resources when you need to know more.
